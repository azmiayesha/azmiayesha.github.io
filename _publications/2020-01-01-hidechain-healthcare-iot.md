Designed and formulated **HIDECHAIN**, a user-centric secure edge computing architecture for healthcare IoT devices to address data security and privacy challenges.

- Analyzed and evaluated the architecture for:
  - Authentication  
  - Authorization  
  - Privacy  
  - Data Integrity  

using **blockchain-based transaction auditing**.
